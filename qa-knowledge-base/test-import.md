# 技术

## Vue 3 的 Composition API 和 Options API 有什么区别？
Composition API 使用 setup() 函数或 `<script setup>` 语法糖来组织逻辑，允许按功能而非选项来组合代码。
Options API 则使用 data、methods、computed 等选项来组织组件逻辑。
Composition API 更适合复杂组件的逻辑复用和类型推导。

## 如何在 Express 中处理文件上传？
可以使用 multer 中间件来处理文件上传。multer 是一个 Node.js 中间件，用于处理 multipart/form-data 类型的表单数据。
配置 multer 时可以指定存储引擎（磁盘或内存）、文件大小限制、文件类型过滤等。

# 生活

## 如何保持每天高效工作？
1. 制定每日计划，列出优先级最高的 3 件事
2. 使用番茄工作法，25 分钟专注 + 5 分钟休息
3. 减少干扰，关闭不必要的通知
4. 定期运动，保持精力充沛
5. 保证充足睡眠

## 新手如何快速上手 Git？
首先学习基本概念：仓库、提交、分支、合并。
常用命令：git init、git add、git commit、git push、git pull、git branch、git merge。
建议通过实际项目练习，从单人开发开始，逐步学习团队协作流程。
