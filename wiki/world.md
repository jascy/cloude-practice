# World Facts

> 青云问道 · 世界观设定

---

## 一、地理格局

### 人界（下界）
- **东荒大陆** — 故事起始之地，灵气稀薄，修仙资源匮乏，散修横行。主角沈青云出生于此。
- **中州** — 修仙文明的核心区域，正道七宗与魔道六派的势力交错之地。灵气浓郁，资源丰富。
- **西域荒漠** — 佛修与苦修士的聚集地，隐藏着上古遗迹。
- **南疆十万大山** — 蛮荒之地，妖兽横行，也是蛮族与巫道的领地。
- **北海冰原** — 极寒之地，传说有上古龙族的遗迹。

### 灵界（上界）
- 飞升之后才能抵达的高等界面，灵气化液，修炼速度是下界的十倍以上。
- 灵界分东西南北中五域，各域由灵王/大乘期修士统治。

### 仙界
- 传说中最高的界面，真仙居所。凡人不可及。

## 二、修炼体系

### 境界划分（炼气 → 大乘）

| 境界 | 寿元 | 特征 |
|------|------|------|
| **炼气期** (1-13层) | 100岁 | 感应天地灵气，引气入体。大多数修仙者终身止步于此。 |
| **筑基期** (初/中/后/圆满) | 250岁 | 筑就道基，灵力固化。可御器飞行，是散修与宗门弟子的分水岭。 |
| **结丹期** (初/中/后/圆满) | 600岁 | 凝结金丹，可凝结丹火炼丹炼器。一方强者，可开宗立派。 |
| **元婴期** (初/中/后/圆满) | 1200岁 | 碎丹成婴，元婴可离体。大修士，在人界为顶级存在。 |
| **化神期** (初/中/后/圆满) | 2500岁 | 元婴化为元神，沟通天地法则。人界罕见，为飞升灵界做准备。 |
| **炼虚期** (初/中/后/圆满) | 5000岁 | 灵界境界。将天地法则炼入己身，虚实相生。 |
| **合体期** (初/中/后/圆满) | 10000岁 | 灵界高阶。身与道合，举手投足引动天地之力。 |
| **大乘期** (初/中/后/圆满) | 理论上无限 | 灵界巅峰。渡过天劫即可飞升仙界。三千年一次大天劫。 |
| **渡劫期** | 过渡阶段 | 大乘圆满后主动引动飞升天劫的阶段，渡过即飞升仙界。 |

### 灵根资质
- **天灵根** — 单一属性灵根，修炼速度极快，万中无一。
- **双灵根** — 两种属性，上等资质，宗门重点培养对象。
- **三灵根** — 三种属性，中等资质，可入宗门内门。
- **四灵根** — 四种属性（主角沈青云的资质），下等资质，修炼速度缓慢。宗门多收为外门杂役。
- **五灵根/杂灵根** — 五种属性俱全，修仙界公认的废灵根。

## 三、功法体系

### 功法等级
| 等级 | 说明 |
|------|------|
| **凡级** | 散修流传的大路货，修炼速度慢、瓶颈多 |
| **灵级** | 小门派镇派功法，有完整的筑基到结丹法门 |
| **玄级** | 中等门派核心功法，可修至元婴 |
| **地级** | 大宗门的根本大法，可修至化神 |
| **天级** | 上古传承，直指大乘。整个修仙界不超过十部 |
| **仙级** | 传说来自仙界，凡人不可得 |

### 功法分类
- **主修功法** — 决定灵力属性和修炼上限
- **炼体功法** — 淬炼肉身，强化防御
- **神识功法** — 修炼神念，用于探查和攻击
- **遁术** — 身法类，用于战斗和逃命
- **秘术** — 燃烧精血/寿元的底牌手段

## 四、丹道体系

### 丹药等级
| 品阶 | 适用境界 | 代表丹药 |
|------|----------|----------|
| **一品** | 炼气期 | 聚气丹、辟谷丹 |
| **二品** | 炼气 → 筑基 | 筑基丹（极其珍贵）、凝气散 |
| **三品** | 筑基期 | 培元丹、洗髓丹 |
| **四品** | 结丹期 | 凝金丹、养魂丹 |
| **五品** | 元婴期 | 化婴丹、天元丹 |
| **六品+** | 化神以上 | 传说中的丹药，灵界才有 |

### 丹师等级
**丹徒 → 一品丹师 → 二品丹师 → ... → 九品丹师 → 丹王 → 丹圣**

炼丹师在修仙界地位尊崇，主角沈青云的主线成长路径之一即为丹道。

## 五、法宝体系

### 法宝等级
| 等级 | 说明 |
|------|------|
| **法器** | 炼气期使用，灵力驱动的简单器物 |
| **灵器** | 筑基期使用，蕴含一丝灵性 |
| **法宝** | 结丹期以上使用，可收入体内蕴养 |
| **古宝** | 上古遗留，威力远超同阶，不可炼制 |
| **通天灵宝** | 化神以上，蕴含天地法则之力 |
| **仙器** | 仙界之物，人界不可见 |

### 主角核心法宝
- **古铜戒指**（初始机缘）— 外表锈迹斑斑的铜戒，内藏一个残破的上古洞天世界。洞天面积随主角修为提升而扩展。其中有一汪灵泉、几亩灵田，以及一座被封印的古老石殿（后期解锁）。

## 六、势力格局（人界）

### 正道七宗
1. **天剑宗** — 剑修第一宗，以攻伐著称
2. **太虚门** — 道门正宗，传承最古老
3. **万法宗** — 以术法繁多著称
4. **青木谷** — 丹道医道，中立势力
5. **御灵宗** — 驭兽之术
6. **天机阁** — 擅长阵法推演，情报交易
7. **浩然书院** — 儒修，读书养气

### 魔道六派
1. **血煞宗** — 血道魔功
2. **鬼灵门** — 御鬼炼尸
3. **合欢宗** — 采补双修
4. **万毒谷** — 毒功蛊术
5. **天魔教** — 最古老魔道宗门
6. **幽冥殿** — 最神秘，行踪不定

### 其他势力
- **散修联盟** — 散修抱团取暖的组织，管理松散
- **坊市商会** — 跨地域的商业网络，以灵石为通货
- **杀手组织** — "影楼"，收钱索命，遍布人界

## 八、特殊资源

- **灵脉** — 灵气汇聚之地，宗门立派的根基。分下品、中品、上品、极品。
- **灵药** — 年份越久越珍贵。百年、千年、万年各有不同药效。
- **灵矿** — 炼制法宝的核心材料。
- **魂兽内丹** — 妖兽体内凝结的灵力结晶，炼丹炼器的重要材料。
